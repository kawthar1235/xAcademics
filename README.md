# KFUPM CRN Schedule & Building Distance Tool

A Java-based academic desktop tool that helps students generate a weekly timetable from class CRNs and estimate the walking distance between consecutive classes on campus.

---

## Overview

The **KFUPM CRN Schedule & Building Distance Tool** is designed to make campus scheduling more practical and efficient for students.  
It allows users to enter their course CRNs, automatically generate a timetable, display class locations by building, and estimate the distance traveled between classes.

This project combines scheduling logic with location-based calculations to support better planning of daily movement across campus.

---

## Features

- Enter course CRNs to generate a weekly timetable
- Display course schedule by day and time
- Show class locations by building
- Calculate walking distance between consecutive classes
- Count the number of courses for a selected day
- Count the number of different buildings visited
- Provide total estimated distance traveled

## How It Works

1. The user enters class CRNs.
2. The system reads and organizes course information.
3. A weekly timetable is generated automatically.
4. Class locations are identified by building.
5. The tool estimates the distance between consecutive classes.
6. A summary is shown, including:
   - number of courses
   - course details
   - number of different buildings
   - total distance traveled

---

## Example Output

For a selected day, the tool can display:

- Selected day
- Number of courses
- Course names and timings
- Number of different buildings
- Estimated walking distance in meters
